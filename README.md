# sistema-gesti-n-incidencias-metro
Sistema desarrollado para el Centro de Control de Fallas del Metro Los Teques (PHP, MySQL)

## ⚠️ Estructura requerida

El sistema espera encontrar los archivos en:
`C:\xampp\htdocs\Programas\Metro\`

La carpeta `Programas` es obligatoria. Sin ella, las rutas fallarán.

## 📥 Instalación rápida

1. Clonar este repositorio o descargar el ZIP
2. Copiar la carpeta `Metro` dentro de `C:\xampp\htdocs\Programas\`
3. Importar `new.sql` a una base de datos llamada `new2` en phpMyAdmin
4. Acceder desde: `http://localhost/Programas/Metro/`
